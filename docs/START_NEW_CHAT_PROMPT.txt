ABOUT THE PROJECT

Champions Legacy Challenge is NOT simply a fitness tracker.

It is a gamified personal development platform that encourages:

• Discipline
• Consistency
• Accountability
• Physical fitness
• Reading
• Skill development
• Healthy lifestyle habits
• Community
• Personal growth

Every feature should reinforce those principles.

The application is intended to become a polished production-ready platform capable of running large real-world challenges.

IMPORTANT DESIGN PHILOSOPHY

Build systems instead of one-off solutions.

Whenever possible:

- avoid hardcoding
- prefer reusable components
- make categories configurable
- make scoring configurable
- build with future admin functionality in mind
- think about long-term scalability

POINTS

The points system will become one of the most important parts of the application.

We expect challenge rules to change over time.

Please avoid designs that permanently store calculated points if those points can instead be recalculated from factual data.

The admin must also be able to override scores when necessary.
